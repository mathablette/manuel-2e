$\Delta = b^2 - 4ac$ est le *{bold::discriminant} du trinôme du second degré $ax^2 + bx+ c$ ($a \not= 0$).
Lorsque le discriminant $\Delta = b^2 - 4ac$ est *{bold::positif} : $\quad\bullet$ L'équation $ax^2 + bx+ c = 0$ possède *{bob1#444444::$2$ racines réelles } $x_1$ et $x_2$. $\quad\bullet$ avec $\displaystyle x_1 = \frac{-b - \sqrt{\Delta}}{2a}$ et $\displaystyle x_2 = \frac{-b + \sqrt{\Delta}}{2a}$
Lorsque le discriminant $\Delta = b^2 - 4ac$ est *{bold::nul} : $\quad\bullet$ L'équation $ax^2 + bx+ c = 0$ possède *{bob1#444444::$1$ racine réelle} $x_0$. $\quad\bullet$ avec $\displaystyle x_0 = \frac{-b}{2a}$
Lorsque le discriminant $\Delta = b^2 - 4ac$ est *{bold::négatif} : $\quad\bullet$ L'équation $ax^2 + bx+ c = 0$ n'admet pas de racine réelle.
Lorsque le discriminant $\Delta = b^2 - 4ac$ est *{bold::positif}, la forme *{bold::factorisée} de $ax^2 + bx+ c$ est alors $a(x-x_1)(x-x_2)$
Lorsque le discriminant $\Delta = b^2 - 4ac$ est *{bold::égale à zéro}, la forme *{bold::factorisée} de $ax^2 + bx+ c$ est alors $a(x-x_0)^2$
Lorsque le discriminant $\Delta = b^2 - 4ac$ est *{bold::négatif}, $ax^2 + bx+ c$ n'a pas de forme factorisée dans $\R$.
Lorsque le discriminant $\Delta = b^2 - 4ac$ est *{bold::positif} ($x_1 < x_2$) alors : $\quad$ Si $a > 0 $ : $\quad\quad\bullet\; ax^2 + bx+ c \geqslant 0$ sur $]-\infty;x_1]$ $\quad\quad\bullet\; ax^2 + bx+ c \leqslant 0$ sur $[x_1;x_2]$ $\quad\quad\bullet\; ax^2 + bx+ c \geqslant 0$ sur $[x_2;+\infty[$ $\quad$ Si $a < 0 $ : $\quad\quad\bullet\; ax^2 + bx+ c \leqslant 0$ sur $]-\infty;x_1]$ $\quad\quad\bullet\; ax^2 + bx+ c \geqslant 0$ sur $[x_1;x_2]$ $\quad\quad\bullet\; ax^2 + bx+ c \leqslant 0$ sur $[x_2;+\infty[$
Lorsque le discriminant $\Delta = b^2 - 4ac$ est *{bold::nul} alors : $\quad$ Si $a > 0 $ : $\quad\quad\bullet\; ax^2 + bx+ c \geqslant 0$ sur $\R$ $\quad$ Si $a < 0 $ : $\quad\quad\bullet\; ax^2 + bx+ c \leqslant 0$ sur $\R$
Lorsque le discriminant $\Delta = b^2 - 4ac$ est *{bold::négatif} alors : $\quad$ Si $a > 0 $ : $\quad\quad\bullet\; ax^2 + bx+ c > 0$ sur $\R$ $\quad$ Si $a < 0 $ : $\quad\quad\bullet\; ax^2 + bx+ c < 0$ sur $\R$
Soit le trinôme ref{t.a}$x^2$ref{t.b}$x$ref{t.c} avec : $a = $ 4 $b = $ 4 $c = $ 4 \\ *{fs14;italic::cliquer sur les coefficients du trinôme pour modifier leur valeur.}
Le discriminant est égal à $\Delta = $ b^2-4*a*c
Comme $\Delta > 0$, il y a deux racines dans $\R$ : tab{2} $x_1$ ref{eq1::$=$} racine1(a,b,c) et $x_2$ ref{eq2::$=$} racine2(a,b,c)
Comme $\Delta = 0$, il y a une seule racine dans $\R$ : tab{2} $x_0 $ ref{eq0::$=$} racine1(a,b,c)
Comme $\Delta < 0$, il n'y a aucune racine dans $\R$.
